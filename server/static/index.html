Timestamp microservice API

A simple api which parses unix timestamps and natural language dates

Example usage

With date: https://time-stampr.herokuapp.com/api/January%201%202018 Go!

Example usage with unix time stamp: https://time-stampr.herokuapp.com/api/1514764800 Go!

Example output

{ unix: 1514764800, natural: "January 1, 2018" }
